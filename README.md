# Calendar App

A mobile-optimized calendar view built with Next.js and TypeScript.

## Features

- Infinite vertical scroll of week rows
- Mobile (iPhone) optimized
- Sticky year/month headers
- Date labels on each day
- Borders around days with thicker borders for month delineation
- Auto-focus on today on page load
- Full-screen layout with sticky toolbar

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app directory
- `components/` - React components
- `components/Calendar.tsx` - Main calendar component
- `components/Calendar.module.css` - Calendar styles
